# Setup

## Virtual Environment
```bash
python -m venv myenv
```

## Activate Environment
```sh
.\myenv\Scripts\Activate.ps1 (powershell)
```

## Git
### Initialise Repository
```bash
git init
```
### Git Ignore
Create `.gitignore` file and add "myenv" to stop tracking that folder's changes

### Add files
1. Stage files -> 
```bash
git add .
```
2. Commit files ->
```bash
git commit -m "message"
```
3. Push to GitHub
